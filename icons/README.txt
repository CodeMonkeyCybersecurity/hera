Please create the following icon files in this directory:

1. icon16.png (16x16 pixels)
2. icon48.png (48x48 pixels)
3. icon128.png (128x128 pixels)

These icons should represent the AuthFlow Inspector extension visually. You can use any image editing software to create these icons.

For a quick start, you can use these free icon resources:
- https://www.flaticon.com/
- https://material.io/resources/icons/
- https://fontawesome.com/icons

Make sure the icons are in PNG format and have transparent backgrounds for the best appearance.
